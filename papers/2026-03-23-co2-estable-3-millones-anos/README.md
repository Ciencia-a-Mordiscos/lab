# El CO₂ No Cambió en 3 Millones de Años y Nadie lo Sabía

Un equipo perforó hielo antártico de 3 millones de años en Allan Hills, Antártida. Las burbujas de aire atrapadas son cápsulas del tiempo que contienen la atmósfera exacta de cuando se formaron.

**El hallazgo:** El CO₂ atmosférico se mantuvo estable alrededor de **250 ppm** durante 3 millones de años. Hoy estamos en **~425 ppm** — un nivel sin precedentes en todo ese registro, a 5.6 desviaciones estándar del promedio.
